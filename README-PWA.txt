Déploiement GitHub Pages :
1. Dézipper ce dossier.
2. Envoyer tout le contenu à la racine du dépôt :
   - index.html
   - manifest.webmanifest
   - sw.js
   - icons/
3. Commit changes.
4. Ouvrir l'URL avec ?v=26 pour contourner le cache.

Kaufler — Portail technicien SAV V2.7

Déploiement GitHub Pages :
1. Dézipper ce dossier.
2. Envoyer tout le contenu à la racine du dépôt :
   - index.html
   - manifest.webmanifest
   - sw.js
   - icons/
3. Commit changes.
4. Ouvrir l'URL avec ?v=27 pour contourner le cache.

Nouveautés V2.7 :
- Liste client intégrée directement dans le HTML.
- Liste N° Affaire + client associé intégrée directement dans le HTML.
- Le champ N° Affaire propose les affaires importées.
- Le client associé se remplit automatiquement lorsqu'une affaire connue est sélectionnée.
- La page Listes accepte le collage : N° affaire + tabulation + client.
